Cryptocurrency Clustering Project

Overview

This project uses machine learning techniques to cluster cryptocurrencies based on market behavior. Key steps include data normalization, dimensionality reduction using PCA, and clustering with K-Means.

Technologies Used

Python

pandas

scikit-learn

matplotlib

hvPlot

Instructions

Install required libraries:

pip install pandas scikit-learn matplotlib hvplot

Run the script:

python main.py

Results

A DataFrame with PCA-reduced data and assigned clusters.

Visualizations of clusters in 2D space using the first two principal components.
